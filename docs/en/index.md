## Start Using

Just a few simple steps to get started with SeeYouUI, see [Installation](/en/install) for more details.

::: code-group

```sh [npm]
npm i see-u-ui

npm i sass@1.63.2 -D
npm i sass-loader@10.4.1 -D
```

```sh [cnpm]
cnpm i see-u-ui

cnpm i sass@1.63.2 -D
cnpm i sass-loader@10.4.1 -D
```

```sh [yarn]
yarn add see-u-ui

yarn add sass@1.63.2 -D
yarn add sass-loader@10.4.1 -D
```

```sh [pnpm]
pnpm add see-u-ui

pnpm add sass@1.63.2 -D
pnpm add sass-loader@10.4.1 -D
```

:::
